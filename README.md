# Sistema de Estoque de Frutas com Array em JavaScript

 Descrição
 --

Nesta atividade vemos a aplicação de array em JavaScript para simular um estoque de frutas.

O sistema permite adicionar, listar e remover frutas dinamicamente, demonstrando na prática a manipulação de arrays utilizando métodos básicos da linguagem.

 Objetivo
 --

Demonstrar o uso de:

Criação de arrays

Manipulação de arrays

Interação com o usuário

Atualização dinâmica de informações na tela

 Funcionalidades
 --

O sistema possui três funcionalidades principais:

1️⃣ Mostrar Frutas
--

Exibe todas as frutas armazenadas no array.

Mostra a quantidade total de frutas.

Informa quando o array está vazio.

2️⃣ Adicionar Fruta
--

Abre uma caixa de diálogo para o usuário digitar o nome da fruta.

Adiciona a fruta ao final do array.

Atualiza automaticamente a exibição.

3️⃣ Remover Última Fruta
--

Remove a última fruta adicionada.

Caso o array esteja vazio, exibe um alerta.

Atualiza automaticamente a exibição.

 Estrutura do Projeto
--

O projeto é composto por:

HTML → Estrutura da página

CSS → Estilização da interface

JavaScript → Lógica de funcionamento do estoque

Todo o código está contido em um único arquivo .html.

 Resultado Esperado
 --

O sistema exibirá:

O array completo

A quantidade de frutas armazenadas

Mensagem informando quando o estoque estiver vazio
